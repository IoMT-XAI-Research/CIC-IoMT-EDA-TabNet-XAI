api.yaml: FastAPI tabanli servis katmaninin ayarlarini tanimlar.
Yani modeli web uzerinden calistiracak sunucunun ayar dosyasi

labels.json: Veri setindeki labellari tanimlar.

mobile.yaml: Flutter uygulamasinin backend API ile iletisimini tanimlar.
Yani mobil istemci icin ag baglanti ve kullanici arayuz parametreleri tanimlar

streaming.yaml: Real-time stream processing yapilandirilmasini yonetir. Gelen veriyi canli isleyip aninda saldiri tespiti yapar.

tabnet.yaml: TabNet modelinin hiperparametrelerini tanimlar. Icindeki degerler modelin ogrenme hizini, genelleme yetenegini ve overfitting riskini dogrudan etkiler.
Yani modelin nasil egitilecegini aciklar
